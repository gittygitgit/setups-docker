Overview

This document describes installing both Docker for Mac and Docker Toolbox.

---
Faq

Why both?
Docker Toolbox support a multi-vm setup.
Docker for Mac is arguably a better container manager, but only supports a single VM.

---

Docker for Mac vs Docker Toolbox
https://nickjanetakis.com/blog/should-you-use-the-docker-toolbox-or-docker-for-mac-windows
https://docs.docker.com/docker-for-mac/docker-toolbox/#the-docker-for-mac-environment

 - Docker Toolbox uses Virtualbox hypervisor
   o Docker machine 
   o supports multiple vm's
 - Docker for Mac uses built in OS/X virtualization 
   o only one vm
